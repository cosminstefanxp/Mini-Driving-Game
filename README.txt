Mini Need For Speed 

The project consists of a basic version of a driving game (eg. Need For Speed), which contains a simple circuit and gives the player full control of the car and the camera. It is developed in Visual Studio, using C++ and OpenGL.

Most important Features: lights, transparency, camera (freecam, follow cam, satellite cam, etc), collision detections, bounding box, side-window containing map view, interactive control of camera using mouse (Shooter style), etc
